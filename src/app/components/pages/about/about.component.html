About this Angular Material Web App Github repo

Things to impress:

  • Completed in just under one week after receiving the brief
  • Search by Kind easily with one click
  • Routing to products, clicking on the product name goes to a details page, go back and the search page is nicely cached (using RouteReuseStrategy from Angular Router )
  • Using Snackbar Material component to "Undo" delete of favourite
  • Fully mobile responsive, no use of Bootstrap, just Material and some Angular Flex Layout
  • Shared functions placed in a Service called InsService, to not repeat code
  • The app meets the demands of the brief, doing the complex add and delete to the Favourites modal
  • Some Unit tests in the Testing folder, done during TDD
